Approximate matrix multiplication [AMM]

Approximate matrix multiplication [AMM]: In this work, I have focused on computing approximate matrix products that are provably accurate. While working on this problem, I came across the applicability of matrix sketching techniques in this task. This led me to develop an AMM algorithm based on the matrix sketching algorithm mentioned earlier (Î²FD). Theoretical guarantees with respect to its accuracy were proved and its superior experimental results over the state of the art was demonstrated.

  • Dataset: Large matrices with and without noise.
  • Implementation done in: Python
  • Software packages used: Scikit-learn, Numpy
