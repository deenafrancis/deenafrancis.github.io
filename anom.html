Kernel based streaming anomaly detection

Identifying anomalous data points from a data stream using the powerful concept of kernels and matrix sketching. The proposed anomaly detection algorithms were demonstrated to be more accurate than the state-of-the-art in practice.

  • Dataset: COD-RNA, Forest, Protein-Homology, MNIST, HTTP, Satellite, Seismic
  • Implementation done in: Python
  • Software packages used: Scikit-learn, Numpy
