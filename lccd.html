Land cover change detection

A land cover change detection algorithm called recursive merge was used to analyze 1 km monthly Enhanced Vegetation Index (EVI) dataset obtained from NASA. OpenCV (C++) was used to work with the dataset. A noise reduction technique based on a statistical hypothesis test was applied on the dataset prior to the application of recursive merge algorithm.

  • Dataset: MODIS EVI
  • Implementation done in: C++
  • Software package used: OpenCV, QT
